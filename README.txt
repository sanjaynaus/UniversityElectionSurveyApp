University Election Survey App
-------------------------------

Description:
This project is a Python-based survey application that helps
students identify the university election candidate that best
aligns with their policy preferences. It conducts a survey,
logs responses, tallies votes, and visualizes results.

How to Run:
1. Ensure Python is installed.
2. Place all files in the same directory.
3. Run the main script:
   python main.py

File Descriptions:
- main.py: Main entry point of the survey application.
- survey_app.py: Contains survey logic and orchestration.
- candidate_data.py: Candidate names and manifestos.
- survey_questions.py: Survey questions.
- data_logger.py: Handles logging of responses.
- visualizer.py: Generates bar and pie charts.

Dependencies:
- matplotlib
-csv
-os
